I Install Ruby(2.3.3)

  WINDOWS
    https://rubyinstaller.org/downloads/

  LINUX
    apt-get install ruby 2.3.3

II Chrome + driver

  install google-chrome
  download appropriate version chromedriver for current google-chrome version
  add path to chromedriver.exe into PATH variable

III Project Bundle

  go to project folder

  type in console:
    gem install bundler
    bundle install

IV Set email and password(sign up before)

  WINDOWS
    set %EMAIL%=your_email
    set %PASSWORD%=your_password

  LINUX
    export EMAIL=your_email
    export PASSWORD=your_password

IV Run test

  type in console:
    cucumber
